Tabs are good to structure your design to not show too much
information at once to the user.

```
<Tabs>
  <Tabs.List>
    <Tabs.Tab>
      Toto
    </Tabs.Tab>
    <Tabs.Tab>
      Tata
    </Tabs.Tab>
  </Tabs.List>
  <Tabs.Panels>
    <Tabs.Panel>
      Toto is beautiful
    </Tabs.Panel>
    <Tabs.Panel>
      Tata is also beautiful !
    </Tabs.Panel>
  </Tabs.Panels>
</Tabs>
```
